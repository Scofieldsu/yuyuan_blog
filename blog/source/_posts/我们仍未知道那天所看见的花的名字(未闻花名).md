```
Secret Base ~君がくれたもの~（秘密基地~你给的东西~）

  --动漫《我们仍未知道那天所看见的花的名字》ED


  君と夏の終わり 将来の夢
  这个与你共度的夏天完结 将来的梦想

  大きな希望 忘れない
  满满的希望 我永远难忘

  10年後の8月 また出会えるのを 信じて
  十年后的八月 请相信我们一定能再见

  最高の思い出を…
  将这最美好的回忆追念

  出会いは ふっとした 瞬間
  初次邂逅 是个不经意的瞬间

  帰り道の交差点で
  在放学归途的十字路边

  声をかけてくれたね 「一緒に帰ろう」
  听见你的一声「一起回家吧」

  僕は 照れくさそうに カバンで顔を隠しながら
  我害羞得手忙脚乱 把脸藏在书包后面

  本当は とても とても 嬉しかったよ
  其实心理 从未如此雀跃欣欢

  あぁ 花火が夜空 きれいに咲いて ちょっとセツナク
  啊 烟花在夜空绽放 几许感伤

  あぁ 風が時間とともに 流れる
  啊 时间像风儿一样 静静流淌

  嬉しくって 楽しくって 冒険も いろいろしたね
  多少开心的相伴 多少欢快的冒险

  二人の 秘密の 基地の中
  都在属于你我的 秘密的基地乐园

  君と夏の終わり 将来の夢
  这个与你共度的夏天完结 将来的梦想

  大きな希望 忘れない
  满满的希望 我永远难忘

  10年後の8月 また出会えるのを 信じて
  十年后的八月 请相信我们一定能再见

  君が最後まで 心から 「ありがとう」叫んでいたこと 知っていたよ
  我知道 直到最后 你都由衷地在心底喊着「谢谢」

  涙をこらえて 笑顔でさようなら せつないよね
  强忍着泪滴 微笑着告别 多么伤心

  最高の思い出を…
  这是我们最美好的回忆
```
